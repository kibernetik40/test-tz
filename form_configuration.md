# Настройка простой управляемой формы для обработки

1. Создайте внешнюю обработку (.epf) в конфигураторе.
2. В модуле обработки вставьте содержимое `processing_module.bsl`.
3. Создайте форму (управляемая форма обработки) с одной кнопкой:
   - Имя команды: "УдалитьНеКлючевые"
   - Подпись: "Удалить не ключевые"
   - Обработчик команды: Вызов процедуры ВыполнитьУдаление(Команда)
4. При желании добавьте область (ТабличноеПоле или Панель) для отображения оставшихся платформ.
   - В коде есть место, где можно вызвать процедуру ПостроитьТаблицуОставшихся(МассивОставшихся) — реализуйте отрисовку под вашу форму.
5. Сохраните обработку и запустите её в тестовой базе.
